# PostgreSQL

Docusaurus can manage multiple versions of your docs.



  
## **Replication**
-------------------
**Description:** Replication refers to any PostgreSQL database system replicating data to another PostgreSQL database system. Example methods are pglogical, OCI GolgenGate, and native functionalities.
